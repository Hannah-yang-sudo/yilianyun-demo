11. 交互规则

所有可点击元素必须有 Hover，导航必须有 Selected / Active，展开收起必须有明确状态。

Do

  • 使用 #2A7DF9 作为主色
  • Hover 使用 #2C87FF 或浅蓝底
  • 表格超宽时仅表格区域横向滚动
  • 操作栏多个操作项之间加分隔线
  • 收起侧栏时内容区同步左移

Don’t

  • 不要重新设计视觉风格
  • 不要在表格 Hover 时改变行高
  • 不要把更多筛选做成弹框
  • 不要卡片套卡片
  • 不要随意新增相近蓝色

  • 搜索区与表格区间距：20px
  • 搜索区文字字号：14px
  • placeholder 颜色：#999999
  • 输入框 / 选择框边框：#E5E6EB
  • 输入框 Hover 边框：#2C87FF
  • 输入框 Hover 背景：#F8FBFF

3. 展开态 / 更多筛选态

点击“更多筛选”后，搜索区向下展开，展示第二行或更多筛选项。

展开态规则：

  • 搜索区高度不固定，根据筛选行数自适应
  • 每一行筛选项高度保持 32px
  • 行间距：16px
  • 筛选项横向间距：16px
  • 第二行筛选项左侧与第一行筛选项左侧对齐
  • 查询 / 重置按钮跟随筛选项自然排列，顺序为先“查询”后“重置”，与前一个筛选项保持 20px 间距
  • 更多筛选 / 收起筛选入口固定在搜索区右侧，不参与筛选项换行
  • 如果筛选项过多，按钮组可以放在最后一行右侧
  • 展开后入口文案变为“收起筛选”
  • 展开 / 收起文字使用主色 #2A7DF9
  • 展开 / 收起箭头跟随状态旋转
  • 展开动画建议 160ms–240ms，不要过度动效

4. 布局规则

搜索区使用 flex 布局或 grid 布局均可，但必须满足：

  • 筛选项按从左到右、从上到下排列
  • 每行筛选项左对齐
  • 按钮组跟随筛选项，不固定到页面最右侧
  • 更多筛选入口固定在搜索区右侧
  • 多行展开时不得打乱查询 / 重置按钮位置
  • 不同页面的筛选项宽度尽量统一
  • 同一搜索区内的普通筛选框长度保持一致
  • 日期范围、长字段筛选可以使用 220px–260px
  • 普通下拉选择使用 160px
  • 关键词输入框可使用 180px–220px

推荐尺寸：

  • 普通 Select / Input：160px × 32px
  • 日期范围：240px × 32px
  • 长关键词输入：220px × 32px
  • 查询按钮：68px × 32px
  • 重置按钮：68px × 32px
  • 更多筛选文字按钮：auto × 32px

5. 交互规则

点击“更多筛选”：

  • 展开隐藏筛选项
  • 箭头向上
  • 文案改为“收起筛选”

点击“收起筛选”：

  • 收起第二行及之后的筛选项
  • 箭头向下
  • 文案改为“更多筛选”

点击“查询”：

  • 使用当前所有筛选条件，包括展开区域中的条件

点击“重置”：

  • 清空所有筛选条件，包括展开区域中的条件
  • 保持当前展开 / 收起状态不强制改变

6. Do

  • 常用筛选项放在第一行
  • 低频筛选项放在展开区域
  • 展开态仍保持搜索区简洁
  • 筛选项宽度尽量统一
  • 查询按钮使用主按钮
  • 重置按钮使用灰色 / 次按钮
  • 更多筛选入口使用文字按钮
  • 展开后行间距保持 16px
  • 搜索区不要压缩表格阅读空间

7. Don’t

  • 不要默认展示所有筛选项
  • 不要让搜索区高度超过页面首屏的 1/4
  • 不要让每个筛选项宽度都不一样
  • 不要把查询 / 重置按钮放到页面中间
  • 不要把更多筛选做成弹框
  • 不要使用新的颜色和圆角
  • 不要在展开区使用卡片嵌套卡片
  • 不要改变已有搜索区的基础风格
